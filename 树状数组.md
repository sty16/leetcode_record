### 树状数组

动态维护序列前缀和(解决区间求和与更新)

+ 单点更新 update(i, v)

+ 区间查询 query(i)

修改和查询的代价都是 O(log$n$) n为所维护序列的长度 

C[i] = A[i - $2^k$ + 1] + A[i - $2^k$ + 2] + $\dots$ +A[i]  k为从二进制中最低位到最高位连续零的长度

```c++
class Solution {
	vector<int> c;
	int n;
public:
	void init(int n) {
		this->n = n;
		c.resize(n, 0);
	}

	int lowbit(int x) {
		return x & (-x);
	}

	void update(int pos, int k) {
		while (pos < n) {
			c[pos] += k;
			pos += lowbit(pos);
		}
	}

	int query(int pos) {
		int ret = 0;
		while (pos > 0) {
			ret += c[pos];
			pos -= lowbit(pos);
		}
		return ret;
	}
};
```
